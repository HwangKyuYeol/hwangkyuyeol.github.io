# 이유식 사이트 상세 레시피 개선 버전

반영 사항:
- 초기 이유식 10배죽 / 7배죽 / 5배죽 농도 가이드 추가
- 상세페이지 만드는 방법을 단계별로 더 자세하게 개선
- 계란 알레르기 안내 강화: 처음에는 완숙 노른자부터 소량 시작
- 알레르기 주의 문구 강화
- 가진 재료 2개 이상 선택 시 레시피가 안 나오는 문제 수정
  - 기존: 선택한 재료가 모두 들어간 레시피만 표시
  - 변경: 선택한 재료 중 하나라도 들어가면 표시 + 많이 일치하는 순서로 정렬
- 기존 기능 유지: SEO, 네이버용 sitemap/robots, 푸터, 문의폼, 상단 이동 버튼
