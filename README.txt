# 이유식 사이트 최종 개선 버전

반영 사항:
- 모바일 카드형 디자인 유지
- 초기 10개 / 중기 30개 / 후기 100개 / 완료기 100개
- 재료 검색
- 재료 선택 메뉴
- 상세페이지 모달
- 광고 위치
- 네이버 SEO용 meta, sitemap.xml, robots.txt
- 예쁜 푸터
- 문의폼: 메일 앱이 열리는 mailto 방식
- 맨 위로 이동 버튼
- 검색창 바로가기 버튼
- 스크롤 방향에 따라 검색 헤더 숨김/노출
- 필수 알레르기 테스트 식품 섹션 추가
- 알레르기 주의 표시 강화
